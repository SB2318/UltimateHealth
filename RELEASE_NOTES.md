# Release Notes v2.2.0

## Features
- Implement article details page with related articles
- Update UltimateHealth Features to 15-point Bento Grid layout
- Redesign platform highlights section with premium aesthetics
- Add auto-scrolling to screenshots carousel
- Add reusable glassmorphic Modal component
- Integrate Knip for unused dependency and dead code detection

## Fixes & Improvements
- Resolve duplicate homepage hero and modal flow
- Preserve signature canvas on resize
- Add `rel="noopener noreferrer"` to external links
- Hide footer social icons when profile URLs are not configured
- Map ambiguous 400 status to generic error in newsletter subscription
- Replace dead anchor tags with Next.js Link component for proper client-side routing
- Apply `withBasePath` to navbar and footer internal links
- Add account deletion routes
- Harden CSP and security headers
